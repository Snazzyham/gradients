Text Gradients

A CSS framework to add gradients to your text

forked from Adam Morse's gradients project.

Get the Code

Reason for existence

I'm a huge fan of Adam Morse's gradients library, but sometimes I want to use gradients on my text and not the background, so I forked Adam's code, added 2 lines of CSS, and made this

A few things to note

  • It only works with webkit
  • It only works good on headers, big headers
  • It will still work with Adam's gradients library, because I've changed the class names from a bg- prefix to a -text suffix.
